# R. Jagadishwar Reddy | Sketchbook Portfolio

A creative, interactive 3D portfolio website that feels like stepping inside a personal sketchbook. Built entirely with vanilla web technologies and Three.js, it combines a hand-drawn 2D doodle aesthetic with 3D depth and exploration.

## 🌟 Features

- **Interactive 3D Hallway:** Navigate through the portfolio by scrolling down a 3D corridor.
- **Hand-Drawn Aesthetic:** All elements feature a sketchy, wobbly hand-drawn style using SVG filters and custom canvas doodles.
- **Darken Mind (Chalkboard Mode):** Toggle between a bright notebook paper theme and a dark chalkboard theme.
- **Floating Skills Workbench:** Real developer icons float dynamically in 3D space, adapting to both light and dark modes.
- **Doodle-Bot Guide:** A custom animated mascot guides visitors through the different sections of the portfolio.
- **Sound Effects:** Pencil scratching sound effects enhance the physical sketchbook feel (toggleable).

## 🛠️ Tech Stack

- **HTML5 & CSS3:** For the structural layout, typography (Google Fonts), and styling (including a CSS-based notebook ring effect).
- **JavaScript (ES6+):** Core logic and application state.
- **Three.js & WebGL:** For rendering the 3D hallway, projecting 2D canvas textures into 3D space, and animating the floating icons.
- **Vite:** Fast development server and bundler.

## 🚀 Running Locally

To run the portfolio on your local machine:

1. **Clone the repository** (or download the source code).
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

This portfolio was designed to be playful and personal rather than corporate. It replaces standard UI elements with doodles, placing personality first and using interactions that reward exploration. The "Darken Mind" toggle is a prime example, completely shifting the visual language from ink-on-paper to chalk-on-slate.
